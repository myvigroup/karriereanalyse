Karrieremesse — Erstkontakt

Touchpoint: Stand, Vortrag, Workshop
🎤 Vortrag / Workshop 💬 Standgespräch 🔍 Pain-Point identifizieren 📱 QR-Code scannen
📊 Ziel: QR-Scan → Landing Page
💎 Lead-Capture
→ "Kostenloser KI-Lebenslauf-Check"
→ QR-Code → Landing Page
→ E-Mail + Name erfassen
1

Lebenslauf-Check — Lead-Magnet

KI-Analyse des CVs mit konkretem Feedback
📄 CV hochladen 🤖 KI-Analyse (/api/analyze-cv) 📊 Ergebnis: Stärken & Schwächen 🔒 Teaser-Ergebnis (ohne Account)
📊 Ziel: Lead → Registrierung
🔓 Absatz-Punkt #1
→ "Volles Ergebnis freischalten"
→ "Was bist du wirklich wert? → Analyse starten"
→ E-Mail-Sequenz bei Nicht-Registrierung
2

Portal-Registrierung & Onboarding

Geführter Einstieg: "Was ist dein Ziel?"
✍️ Account erstellen 🎯 Ziel auswählen 👋 Welcome-Notification 📊 Dashboard
💰 "Mehr Gehalt"
→ Analyse + Gehaltsverhandlung
📋 "Bewerben"
→ CV-Check + Kanban
🤔 "Bleiben/Gehen?"
→ Entscheidungs-Kompass
📊 Ziel: Registrierung → Erste Aktion
🎯 Absatz-Punkt #2
→ Personalisierte Tool-Empfehlung
→ Zielbasierter Einstieg statt Überflutung
→ Aha-Moment in < 5 Minuten
3

Aktivierung — Erste Wertschöpfung

Kostenlose Tools liefern echten Mehrwert → Kaufbereitschaft
📈 Karriereanalyse (65 Fragen) 🧭 Entscheidungs-Kompass 💹 Marktwert-Cockpit 🤖 KI-Coach Chat 💰 Gehaltsdatenbank
Analyse → PRIO
"Deine #1 Priorität: Gehaltsverhandlung"
Kompass → Richtung
Gehen → Bewerbung / Bleiben → Verhandlung
Marktwert → Lücke
"Du lässt €X.XXX liegen"
📊 Ziel: Aktiver Nutzer → Masterclass-Buchung
🔥 Absatz-Punkt #3
→ Analyse-Ergebnis → Kurs-Empfehlung
→ Gehaltslücke → "Lerne verhandeln"
→ KI-Coach empfiehlt Masterclass
→ Impostor-Score → Confidence-Kurs
4

Masterclass — Monetarisierung

Kostenpflichtige Kurse (13 Masterclasses verfügbar)
🎬 Video-Lektionen 📝 Übungen & Skripte ⭐ XP & Level-Ups 🏅 Badges & Zertifikate 💹 Marktwert steigt sichtbar
📊 Ziel: Kurs abschließen → Cross-Sell
🔄 Absatz-Punkt #4
→ Cross-Sell: "Nächster Kurs: LinkedIn-Branding"
→ Bundle: "Alle 13 Kurse zum Vorteilspreis"
→ Zertifikat: "Noch 2 Kurse bis Certified Pro"
5

Umsetzung — Verhandlung & Bewerbung

User setzt das Gelernte in der Praxis um
📋 Bewerbungs-Kanban nutzen 🎯 Interview-Briefing generieren 💰 Gehalt verhandeln 📓 Gehalts-Tagebuch führen 🚪 Exit-Strategie planen
📊 Ziel: Erfolg erzielen → Premium-Upsell
⚡ Absatz-Punkt #5
→ 1:1 Coaching vor der Verhandlung
→ Exit-Strategie-Paket (Premium)
→ Peer-Matching: Accountability Partner
6

Alumni — Retention & Wachstum

Ziel erreicht — langfristige Bindung & Weiterempfehlung

📊 Ziel: Retention + Empfehlung + B2B
